##  Why this project

I built this to practice building a modern, component-based frontend with React and to create a nice-looking, easily customizable coffee shop page.  
It’s ideal as a portfolio project, a base for a small business website, or for learning React + styling fundamentals.

##  Tech Stack & Tools

- React (via Create React App)  
- JavaScript (ES6) + JSX  
- HTML5 
- CSS3 for layout, typography, responsiveness  
- Browser-based — runs locally 

##  Features (so far)

- Header + Navigation  
- Hero section with background images
- Menu section listing coffee items
- Footer
- Clean, modern design — styled with custom CSS
- Responsive / easy to modify

##  Status
Project is functional and ready to use as a static website template.
